# Maho Maths

A fun and interactive addition and subtraction application designed for Grade 1 learners. All answers are less than 20. Works fully **offline** and can be installed on phones and tablets as an app.

## Features

- Addition, Subtraction, and Mixed mode
- Dot visualizer to help kids count along
- 4 colourful answer buttons per question
- Live score tracking with stars
- Confetti celebration for correct answers
- 10-question rounds with a results screen
- Works offline after the first visit
- Installable as an app on Android, iPhone, and desktop

## Files

| File | Description |
|------|-------------|
| `index.html` | The main application |
| `sw.js` | Service worker — enables offline use |
| `manifest.json` | PWA manifest — enables "Add to Home Screen" |
| `icon.svg` | App icon |
| `README.md` | This file |

## Installing as an App on Phone

Once the site is live:
- **Android**: Open in Chrome → tap the three-dot menu → "Add to Home Screen"
- **iPhone**: Open in Safari → tap the Share button → "Add to Home Screen"
